Software / Tools

  • 2025 – ANASOP – Análisis de Operaciones: Python / Educational Engineering Tool. Simulates a production line with four independent stations, adjustable production times, lot sizes, and inventory levels. Computes key metrics including lead time, cycle time, and throughput.
    Co-authors: Carlos Eduardo Montoya Casas, Luis Andrés Saavedra Robinson.
    Copyright: Pontificia Universidad Javeriana, Colombia.
  • 2020 – INGEPROtool: Microsoft Excel VBA Add-in with Custom Ribbon / Educational Engineering Tool. Provides an interactive platform for visualizing engineering process concepts, including facility layouts (CORELAP, CRAFT), Critical Path Method, and Rank Order Algorithm.
    Co-authors: Andrés Felipe Yaya Bohorquez, Harry Hernandez Serrato, Luis Andrés Saavedra Robinson, Pawel Breznev Alayon Benavides, Alexandra Cabanzo Moisés.
    Copyright: Pontificia Universidad Javeriana, Colombia.
  • 2011 – FODESEPProspec: Microsoft Access VBA / Cost Prediction Software. Estimates annual demand costs for higher education institutions to support quality, coverage, and relevance targets.
    Copyright: Higher Education Development Fund – FODESEP. Website
  • 2011 – MECET System: Microsoft Access VBA / Growth Assessment and Location Planning Model. Estimates and analyzes urban development budgets based on comparable projects, supporting strategic planning decisions.
    Copyright: Fundación AMANS URDEM.
